How it runs

This site is a status dashboard. The checker itself runs on a schedule via GitHub Actions every 2 hours.

Gmail is supported out of the box. Use these secrets:

  • GEMINI_API_KEY
  • SMTP_HOST = smtp.gmail.com
  • SMTP_PORT = 587
  • SMTP_USER = your Gmail address
  • SMTP_PASS = Gmail App Password
  • EMAIL_FROM and EMAIL_TO
